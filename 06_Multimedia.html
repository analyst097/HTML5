Multimedia in a webpage

Objectives

  • Describe HTML audio and video elements
  • Explain the purpose of plugins
  • Reflect on accessibility issues

HTML5 Multimedia

  • Designed to avoid any external plugins to play audio/video
  • Not fully implemented

Video element <video>

  • Video tag uses a src attribute or embedded <source>
  • Common attributes
    • height, width
    • autoplay
    • loop
    • controls
  • Text inside <video> is displayed if browser cannot support tag

Audio element <audio>

  • Uses a src attribute to link to .mp3 or .wav file
  • Common attributes
    • autoplay, loop, controls
    • buffered
    • muted
    • volume

Setting clips

  • Play clips of video and audio elemets by adding to the src attribute:
    • .ext#t= 5, 25
    • .ext#t= , 39
    • .ext#t= , 1:29:23
    • .ext#t= 42
