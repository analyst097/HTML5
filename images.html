Image tag

It's better to modify the actual image file if possible rather modifying width and height.
Reducing size of the actual image reduces the file size which leads to faster loading of web page.
It's also helpful in displaying accurate size of image we need.

Alt attribute

  • Provides textual alternative to non-text content.
  • Read by screen readers
  • Displayed in place of images
  • Provides semantic meaning for search engines.

Creating good alt text.

  • be accurate
  • succint
  • Don't be redundant
  • Don't include "picture of..", "graphic of.."
  • It's ok to have empty alt text
  • Do not skip it though as it helps in accessibility.
